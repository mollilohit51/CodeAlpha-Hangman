# CodeAlpha- Hangman Game

# Description
A simple text-based Hangman Game built using 
python for the CodeAlpha Python Programming Internship.

In this game, the player attempts to guess a 
hidden words one letter at a time. The player has 
a unlimited number of incorrect guesses to 
identify the word before the game ends.

# Features
- Random word Selection
- Letter-by-letter guessing
- Limited incorrect attempts
- Win and lose conditions
- Console-based gamplay
  
# Technologies used
- Python

# Concepts Used
- Loops
- Conditional Statements
- Lists
- Strings
- Random Module

# How to Run
1. Install Python
2. Open terminal or command prompt
3. Run the file:

python Hangman.py
